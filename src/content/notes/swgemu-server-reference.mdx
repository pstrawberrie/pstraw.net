### Docker Examples

Docker compose attach to `swgemu` image:
```bash
sudo docker compose attach swgemu
```

Docker login to db image as user `swgemu`
```bash
sudo docker exec -it db mariadb -u swgemu -p
```

### SQL Examples
Basic syntax reminders:
```sql
use swgemu; -- use the database first
show tables; -- display existing tables
select * from tablename; -- show data from table
```

Set admin level for user `pstraw`:
```sql
update accounts set admin_level = 15 where username = 'pstraw';
```

### Linux General
Screen syntax:
```bash
screen -ls # list all screens
screen -S hello # start new screen with name 'hello'
screen -r hello # attach to existing screen 'hello'
# [CTRL+A, D] detaches from screen
```
